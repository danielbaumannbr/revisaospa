1 O que é o Framework?

O Angular é uma plataforma de desenvolvimento e um framework baseado em TypeScript. Ele é projetado para criar aplicações de página única (SPA - Single Page Applications) eficientes e complexas.

Dica: Diferente de bibliotecas como o React, o Angular é um framework "batteries-included", oferecendo roteamento, gerenciamento de formulários e cliente HTTP nativamente.

2 Arquitetura de Componentes

A estrutura central do Angular baseia-se em componentes. Imagine que cada parte da sua interface (menu, lista, rodapé) é um bloco independente.

HTML

Define o esqueleto e a estrutura da interface (Template).

TypeScript

Contém a lógica de negócio e o comportamento do componente.

CSS/SCSS

Define o estilo visual específico daquele componente.

3 Como começar (CLI)

01.

Instale o Angular CLI

Abra seu terminal e instale a ferramenta de linha de comando globalmente:

npm install -g @angular/cli
02.

Crie um novo projeto

O CLI criará toda a estrutura de pastas necessária:

ng new meu-primeiro-app
03.

Inicie o servidor

Entre na pasta e rode o projeto em tempo real:

cd meu-primeiro-app
ng serve --open
